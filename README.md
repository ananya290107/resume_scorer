# AI Resume Scorer
This is an AI-based tool that analyzes resumes and gives feedback.

You can upload your resume (PDF or Word) and optionally a job description.  
The AI will then:

- give an overall score
- check ATS friendliness
- find missing keywords
- list strengths and weaknesses
- suggest improvements
- rewrite weak bullet points

This project was created for learning purposes and to help improve resumes.

## How to run
1. Install Python 3.9 or newer

2. Install required packages

3. Add your OpenAI API key as an environment variable  
Windows:set OPENAI_API_KEY=your_key_here
Linux:export OPENAI_API_KEY=your_key_here

4. Run the app

## Notes
- Do not upload sensitive personal data to public repositories
- This tool does not guarantee any job or ATS pass
- It is meant for practice and learning
